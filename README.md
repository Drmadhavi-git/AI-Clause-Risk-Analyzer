# AI Clause Risk Analyzer

An end-to-end AI system that extracts clauses from procurement and contract PDFs, classifies their risk level, and suggests mitigation strategies using Large Language Models.

## Features
- PDF clause extraction
- Clause-level risk classification (Legal, Financial, Compliance)
- AI-generated risk explanation
- Suggested mitigation actions
- API-ready architecture

## Tech Stack
- Python
- LangChain
- OpenAI / HuggingFace LLMs
- FAISS
- Flask
